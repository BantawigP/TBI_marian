# Features Folder

This folder is the new feature-first structure.

- `events/` is the first migrated template feature.
- `auth/`, `contacts/`, `incubatees/`, and `team/` are scaffolded for incremental migration.

Recommended layout per feature:

- `components/`
- `pages/`
- `hooks/`
- `services/`
- `types.ts`
